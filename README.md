# API-topskor-liga

Penggunaan API untuk cek Top-Skor Liga dengan AJAX

Fitur Liga :

<ul>
<li>Liga Champions</li>
<li>Liga Inggris</li>
<li>Liga Italia</li>
<li>Liga Spanyol</li>
</ul>

Fitur Statistik :

<ul>
<li>Tokskor</li>
<li>Assist</li>
</ul>


Parameters :
Liga
- ?liga=champions
- ?liga=inggris
- ?liga=spanyol
- ?liga=italia

Statistik
- ?stat=topskor
- ?stat=assist

ApiKey
- ?apikey=YOUR API KEY

Contoh : 
http://bayuu.net/api/cek-topskor?liga=spanyol&stat=topskor&apikey=MASUKAN_API_KEY_ANDA
